# pydantic-ai
Learn how to build AI agents with PydanticAI

## Tech Stack

- Python
- PydanticAI
- Ollama
- Pydantic

## Getting Started

1. Clone the repository.
2. Install the required dependencies.
3. Install and start Ollama.
4. Pull an open-source model, for example:

```bash
ollama pull llama3.2
```

5. Run the example scripts and start experimenting with PydanticAI agents.

## Notes

This repository is intended for learning and experimentation. As I continue exploring PydanticAI, more examples and agent implementations will be added.
